This folder contains analysis scripts and results from independent component analysis of resting state data using the GIFT toolbox in SPM.
This is the second ICA analysis, updated at the request of a reviewer, and presented as main results in the associated paper.
The file "Input_data_subjects_2.m" was used to enter data, and contains the settings used. The sections specifying files to be read have been redacted in order to preserve participants' anonymity.
Analysis output files are also made available here, allowing for inspection of networks.
